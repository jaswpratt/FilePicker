# scotch-face-mash
Filepicker demo app to build a wall of user uploads.

## Instructions

1. Clone repo: `git clone git@github.com:scotch-io/scotch-face-mash`
2. Install dependencies: `npm install`
3. Start the server: `node server.js`
4.
